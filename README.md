# Retail Inventory Management System with OpenFoodFacts Integration

A Flask-based REST API featuring mock array storage, seamless validation parsing data structures from OpenFoodFacts via barcode inputs, an easy-to-use CLI Frontend application, and structured testing patterns.

## Initial Setup
1. Install requirements: `pip install flask requests pytest`
2. Spin up the Flask API: `python app.py`
3. Launch the operational workspace interface terminal in another window: `python cli.py`
4. Execute validation testing patterns: `pytest test_app.py`
